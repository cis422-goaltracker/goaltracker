# University of Oregon CIS 422:
## Goal Tracker

### Supported Operating Systems
* Windows 10 (18.03)
* Mac OSX (10.14.3)
* Ubunutu (18.04 LTS)
* Debian (9.8)
* Arch (2019.03.01)
* Fedora (29 Workstation)
* Chrome OS (72.0.3626.117)
### Installation
```sh
$ pip3 install pyqt5
$ pip3 install DateTime
$ pip3 install pygraph
```
### Running the Script
```sh
$ python3 ViewController.py
```
